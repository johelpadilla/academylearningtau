# Academy Learning Tau

**Plataforma educativa y de investigación** — Tau Sistémica + RECD.

Este repositorio es el producto **nuevo** de la app en desarrollo  
(`systemic-tau-platform` / localhost). **No** es una app antigua de Antigravity ni de `systemictau` (PyPI).

## Web

- **Pública:** https://academylearningtau.streamlit.app  
- **Main file (Streamlit Cloud):** `app/streamlit_app.py`  
- **Branch:** `main`

## Local

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
streamlit run app/Home.py
```
